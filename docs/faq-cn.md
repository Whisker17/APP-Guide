# FAQ

- **如何确认自己购买成功？**

  1. 查询自己的交易是否成功

     https://kusama.subscan.io/account/

  2. 通过 RMRK 的网站查询自己的地址是否存在 NFT

  3. 如果没有，可能是 RMRK 网站还没有完成快照，可以通过下面这个网址查询是否已经铸造：

     https://rmrk-check.zeitgeist.pm/valid/地址   （地址替换成本人地址）

     `true` 为已经铸造， `false` 为还未铸造
